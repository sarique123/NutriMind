# 🧠 Lifestyle Recommender for College Students

An AI-powered lifestyle and wellness recommendation system designed for college students using React, FastAPI, TensorFlow, and Machine Learning.

---

# 📌 Project Overview

This project helps students improve their daily lifestyle habits by analyzing:

- 😴 Sleep quality
- 📚 Study load
- 🤕 Headache frequency
- 🏃 Extracurricular activities
- 🍲 Food preferences and ingredients

The system provides:

- 🧠 Stress level prediction using Deep Learning
- 🍽️ Personalized food recommendations using Machine Learning
- 🌐 Responsive frontend with React

---

# 🧩 Features

## 🧠 Stress Prediction System
- Predicts student stress levels
- Uses Deep Neural Network model
- Multi-level stress classification:
  - Low
  - Medium
  - High
  - Very High
  - Extreme

---

## 🍲 AI Diet Recommendation System
- Ingredient-based recipe filtering
- Veg / Non-Veg filtering
- Exclude unwanted ingredients
- KNN-based recommendation engine
- Nutrition similarity matching

---

## 🌐 Frontend Features
- Responsive modern UI
- Multiple pages using React Router
- Lifestyle form handling
- Contact / Feedback form
- Interactive stress prediction interface

---

# ⚙️ Machine Learning Pipeline

## Stress Prediction Model
1. Dataset preprocessing
2. Feature scaling using StandardScaler
3. Deep Neural Network training
4. Model evaluation
5. Model serialization using `.keras`

---

## Diet Recommendation Engine
1. Ingredient filtering
2. Nutrition preprocessing
3. Standard scaling
4. Cosine similarity calculation
5. KNN recommendation generation
